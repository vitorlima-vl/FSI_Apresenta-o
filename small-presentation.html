O que são dados? (1/2)

  • Dados são observações documentadas ou resultados de uma medição
  • Para a informática, os dados são expressões gerais que descrevem características das entidades sobre as quais operam os algoritmos

Pode-se entender um dado como um elemento da informação (um conjunto de letras ou dígitos) que, tomado isoladamente, não transmite nenhum conhecimento, ou seja, não contém nenhum significado intrínseco

O que são dados? (2/2)

Oportunidades nos dados

  • Houve uma evolução drástica da infraestrutura para armanezagem e coleta de dados nos últimos 15 anos
  • Utilizar estes dados de alguma maneira passou a ser uma vantagem para as organizações
  • Previsão
  • Probabilidade e Risco

Análise de dados

  • A formalização do processo (semi)automatizado de análise de dados tem recebido diferentes nomes ao longo dos anos
  • Mais recentemente, o processo de análise de dados tem sido conhecido
    como Data Science

Por fim, o que é informação?

  • A informação deve ser completa, confiável, relevante, veríficavel, acessível e, uma vez que traz vantagem competitiva, segura
  • Para melhor uso da informação na tomada de decisão, é importante que ela esteja atualizada, cubra uma area signficativa e seja disponibilizada para as pessoas corretas

Informação é a resultante do processamento, manipulação e organização de dados, de tal forma que represente uma modificação no conhecimento do sistema que a recebe
