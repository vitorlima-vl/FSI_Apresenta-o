agosto – 2017

O que são dados? (1/2)

  • Dados são observações documentadas ou resultados de uma medição
  • Para a informática, os dados são expressões gerais que descrevem características das entidades sobre as quais operam os algoritmos

Pode-se entender um dado como um elemento da informação (um conjunto de letras ou dígitos) que, tomado isoladamente, não transmite nenhum conhecimento, ou seja, não contém nenhum significado intrínseco

O que são dados? (2/2)

Dilúvio de dados

  • Houve uma evolução drástica da infraestrutura para armanezagem e coleta de dados nos últimos 15 anos
  • A evolução da tecnologia influenciou fortemente este processo
  • Mais do que nunca, coletar e analisar dados passou a ser uma necessidade das organizações

Oportunidades nos dados

  • Previsão
  • Risco e Probabilidade
  • Recomendações
  • Localização de sequências
  • Agrupamento

O que é informação? (1/2)

  • A formalização do processo (semi)automatizado de análise de dados tem recebido diferentes nomes ao longo dos anos
  • Muitos dos métodos classificados como de Data Mining, também são classificados como de Aprendizado de Máquinas
  • Há ainda interseções (ou sobreposições) com Business Intelligence, que muitas vezes inclui métodos de data mining/machine learning e outras técnicas da área de Banco de Dados
  • Mais recentemente, o processo de análise de dados tem sido conhecido como Data Science

O que é informação? (2/2)

  • Dado preciso, específico e organizado para um propósito, apresentado dentro de um contexto que lhe dê significado e relevância
  • A informação deve ser completa, confiável, relevante, veríficavel, acessível e, uma vez que trazem vantagem competitiva, segura
